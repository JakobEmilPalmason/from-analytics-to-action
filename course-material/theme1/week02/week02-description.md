### Week 2 (9 February) - Datadriven ambitions and their organisational implementation

The topic of this class is the relation between technology and organisation. This is a relationship that has been covered in the Organisation Studies literature already before digital technologies and big data started to play a role for operations and strategy. Today’s main text describes how organisations change when digital technologies acquire a main role in the day-to-day work. The text presents five ways in which technology afford new kinds of behaviour. Affordance is a key term offered by the text. You can use it to understand how data and data projects afford organisational change. The idea that technology affords organisational change raises the question if organisations afford changes in technology.
